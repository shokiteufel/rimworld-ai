# Stories

Dieses Verzeichnis wird per Sprint gefüllt. Stories werden aus `../../_bmad-output/planning-artifacts/epics.md` gezogen und hier einzeln gedraftet.

**Format:** `N-M-kebab-case-name.md` (z. B. `1-1-mod-project-skeleton.md`)

**Status-Convention** (Quelle: `../sprint-status.yaml`):
- `backlog` — Geplant, nicht drafted
- `ready-for-dev` — Drafted, Akzeptanzkriterien approved, bereit für Dev
- `in-progress` — Aktuell in Arbeit
- `review` — Fertig, wartet auf Code-Review / Visual-Review
- `done` — Reviews passed, Story abgeschlossen
- `optional` — Nice-to-have, nicht Launch-critical

Stories werden erst gedraftet, wenn PRD und Architecture approved sind.
